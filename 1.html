Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.
Web browsers receive HTML documents from a web server of from local stroage standard and render them into multimedia web pages.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other obfects, such as interactive forms, may be embedded into the rendered page.
